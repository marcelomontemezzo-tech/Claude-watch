# Clients

Um arquivo `.md` por cliente. Cada arquivo é puxado como contexto quando o agent toca uma feature ligada àquele cliente.

Esquema sugerido:

```markdown
---
cliente: ZapLub
projeto: torque
ativo: true
---

## Quem é
[descrição curta]

## Particularidades
- [regra de negócio específica]

## Pessoas
- [nome] — [papel]
```
